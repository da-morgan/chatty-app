** ChattyApp **
=====================

ChattyApp is a chat application that allows connected users to create a username
and send each other messages. 

### Usage

Install the dependencies and start the servers.

```
npm install
npm start
open http://localhost:3000
```

### Dependencies

* React
* Webpack
